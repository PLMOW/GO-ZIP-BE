## 구현기능 
- 스프링 시큐리티를 이용한 JWT 로그인  
- 게시글 CRUD  
- S3를 이용한 이미지 저장기능  
- QueryDSL을 이용한 동적인 조건부 검색기능  
- 예외처리  
  - GlobalExceptionHandler로 controller의 예외를 전역적으로 처리  
  - ExceptionHandlerFilter를 추가하여 시큐리티에서 JWT검증간의 예외를 일괄적으로 처리
